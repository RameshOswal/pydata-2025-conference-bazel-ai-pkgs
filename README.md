# Building Bazel Packages for AI/ML: SciPy, PyTorch, and Beyond

This repository contains the materials and code examples from the PyData Seattle 2025 conference presentation on building Bazel packages for AI/ML libraries.

## Abstract

This presentation explores the process of building and managing Bazel packages for popular AI/ML libraries including SciPy, PyTorch, and other essential tools in the data science ecosystem.
